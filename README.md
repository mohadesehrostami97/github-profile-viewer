A simple web app that allows you to search GitHub users and view their profile information using the GitHub API.  
It also supports **Dark / Light mode toggle** for better user experience.

---

# Features

- Search GitHub users by username  
- Display user avatar, name, bio, and location  
- Show number of repositories, followers, and following  
- List latest public repositories  
- Dark / Light mode support  
- Basic error handling (user not found, API limit, etc.)
